# Lazy Invoice

> 輕鬆自在開發票

![Lazy Invoice 歡迎畫面](assets/images/screenshots/intro.png)


**Lazy Invoice 的特色**：
- **平台支援**：支援 ECPay、ezPay、O'Pay、SmilePay、AMEGO 五家加值中心
- **發票開立**：使用 AI 工具協助處理，或在 UI 上手動新增
- **開立模式**：支援正式及測試兩種模式
- **彈性部署**：線上版純 JS，自架版需 PHP 環境
